v6 更新内容

・不正解音を、はっきり聞こえる2段階のブザー音に変更
・iPhoneで押し間違えないよう、「終了」を問題画面の右上へ移動
・「次へ」を大きく独立表示
・回答後に「できなかったことにする」ボタンを追加
・偶然正解した単語も、間違いリストと今回の復習リストに登録可能
・その場合、正解数も1つ戻して正答率を調整
・全1671語／1級634語の切替など、従来機能は維持

更新方法
1. ZIPを解凍
2. GitHubの topik-1671 を開く
3. ＋ → Upload files
4. index.html と service-worker.js をアップロード
5. Commit changes
6. 2～3分待つ
7. Safariで ?v=6 を付けて開く
   https://akipanda5highfive.github.io/topik-1671/?v=6
8. 動作確認後、ホーム画面アプリを閉じて開き直す
