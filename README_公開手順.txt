【大事】
iPhoneの「ホーム画面に追加」を使うには、このフォルダ一式を
HTTPS対応のWebサービスに公開する必要があります。
ファイルアプリから直接開いただけでは、ホーム画面追加はできません。

【いちばん簡単な公開方法：GitHub Pages】
1. GitHubで無料アカウントを作る
2. 新しいリポジトリを作る
3. このフォルダ内のファイルを全部アップロードする
4. Settings → Pages
5. Branchを main / root にしてSave
6. 表示されたURLをSafariで開く
7. 共有ボタン → ホーム画面に追加

【注意】
学習記録は、公開先URLごとのSafari内に保存されます。
公開先URLを変えると、別の記録として扱われます。
