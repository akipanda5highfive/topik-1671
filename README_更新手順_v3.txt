GitHub更新方法
1. 更新用ZIPを解凍
2. GitHubの topik-1671 を開く
3. Go to file の右の「＋」→ Upload files
4. index.html と service-worker.js をアップロード
5. Commit changes
6. 1～3分待つ
7. iPhoneのアプリを完全に閉じ、もう一度開く

鳴らない場合
・iPhoneの消音モードを解除
・音量を上げる
・最初の回答は画面をタップして行う
